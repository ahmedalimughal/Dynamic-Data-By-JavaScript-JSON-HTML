# Dynamic Data By JavaScript JSON HTML

This project allows you to:

- Add data using a Bootstrap form.
- Save all entries in a single `data.json` file via PHP.
- Display the data as cards using HTML, Bootstrap, and JavaScript.
- Use live search and pagination (9 cards per page).

## 📁 Features

✅ Add data: image, heading, details, date, and link  
✅ Save to one JSON file (`data.json`)  
✅ Display as responsive Bootstrap cards  
✅ JavaScript-based live search  
✅ Pagination (9 items per page)  

## 📂 Folder Structure

project/
├── form.html # Form to add entries
├── display.html # Card view with search + pagination
├── save_data.php # Handles form submission
├── data.json # Stores all data in JSON
└── uploads/ # Stores uploaded images


## 🚀 How to Use

1. Open `form.html` in browser to add data.
2. Visit `display.html` to view saved data.
3. Make sure your server supports PHP and write access.

## 💡 Built With

- HTML5, CSS3 (Bootstrap 5)
- JavaScript (vanilla)
- PHP
- JSON
